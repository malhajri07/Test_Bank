## How to use this file

- **Adding a finding:** add a row to "Still open", don't create a new `*.md`.
- **Landing a fix:** move the item from "Still open" to "Applied" with the file refs; don't write a new `FIXES_COMPLETE.md`.
- **Stale section:** if something's been in "Still open" unchanged for >1 quarter, either delete it (not a real priority) or bump it up.
